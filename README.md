# HH.ru Resume Free Up

Бесплатный инструмент автоматизации для соискателей на hh.ru. Расширение разработано для тех, кто хочет держать свое резюме на верхних строчках поисковой выдачи рекрутеров, поддерживать высокую органическую активность профиля и экономить время на рутинном просмотре вакансий.

## Зачем это нужно?

Алгоритмы hh.ru устроены так, что рекрутеры видят первыми те резюме, которые были обновлены недавно. Кроме того, активность соискателя на сайте (просмотры вакансий, добавление в избранное) положительно влияет на внутренний рейтинг профиля. 

Делать это вручную каждые несколько часов — рутина. **HH.ru Resume Free Up** берет эти задачи на себя: расширение работает незаметно в фоне прямо в вашем браузере, имитируя поведение реального человека и освобождая ваше время для подготовки к собеседованиям.

---

## Основные возможности

* **Автоподнятие резюме 24/7:** Автоматически нажимает кнопку «Поднять в поиске».
* **Умная имитация активности:** Имитирует живой интерес кандидата, просматривая подходящие вам вакансии.
* **Автоматический отбор по навыкам (лайки):** Анализирует требования вакансий по отношению к вашему резюме. Если процент соответствия ключевых навыков превышает ваш личный порог (например, 60%), вакансия автоматически сохраняется в Избранное.
* **Быстрая пауза:** Кнопка мгновенной приостановки на главном экране позволяет в один клик остановить все фоновые процессы, когда вы хотите поработать с hh.ru вручную.
* **Локальная безопасность:** Все данные (резюме, настройки, кэш просмотренного) хранятся исключительно внутри вашего браузера. Расширение не использует сторонние сервера и не передает личные данные.

## Установка в браузер

1. Скачайте файлы проекта в удобную папку на компьютере.
2. Перейдите в браузере Google Chrome (или другом браузере на базе Chromium) по адресу `chrome://extensions/`.
3. Включите **«Режим разработчика»** (Developer mode) в правом верхнем углу страницы.
4. Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked) в левом верхнем углу.
5. Выберите папку с проектом (где находится файл `manifest.json`).

## Быстрый старт

1. Откройте вкладку со своим резюме на hh.ru (поддерживаются любые региональные поддомены, например `spb.hh.ru`). </br></br><img width="645" height="1214" alt="resume" src="https://github.com/user-attachments/assets/b70a65e2-de20-4984-a3cd-757d7aa90b4e" />
2. Кликните на иконку расширения **HH.ru Resume Free Up** на панели браузера. </br></br><img width="494" height="310" alt="image" src="https://github.com/user-attachments/assets/95c12b38-e711-43d0-b17f-15e68d354849" />
3. Нажмите кнопку **«Добавить текущее»** — расширение автоматически импортирует ID резюме, его название и список ваших навыков для фильтрации. </br></br><img width="509" height="614" alt="image" src="https://github.com/user-attachments/assets/bd46e5c0-920b-4af0-8d91-2633d7010a15" />
4. Задайте желаемый интервал проверки (по умолчанию 4 часа) и критерии соответствия вакансий в процентах. 
5. Нажмите **«Сохранить»**. </br></br><img width="494" height="401" alt="image" src="https://github.com/user-attachments/assets/a17470e9-dd6d-4e9c-babf-ea2ae6af9760" />
